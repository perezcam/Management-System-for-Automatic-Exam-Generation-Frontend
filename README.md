# Frontend — Gestión Automática de Exámenes (UI)

Stack: **Next.js + React + TypeScript + TailwindCSS + shadcn/ui (Radix Primitives)**

> Este README te guía para **instalar dependencias**, **correr el proyecto** y **añadir los paquetes de Radix** que usan los componentes del directorio `./components/ui`.

---

Comprueba versiones:
```bash
node -v
npm -v
```

Instala dependencias:
```bash
npm install
```

```bash
npm install @radix-ui/react-avatar @radix-ui/react-checkbox @radix-ui/react-dialog @radix-ui/react-label @radix-ui/react-separator @radix-ui/react-switch @radix-ui/react-slot
```

Opcional si es necesario:
```bash
npm install @radix-ui/react-icons
```

Para correr el proyecto:
```bash
npm run dev
```
